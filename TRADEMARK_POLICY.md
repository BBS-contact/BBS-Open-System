# Trademark Policy

This document defines the trademark and brand usage rules
for Fundacja BBS — Better Balance System.

---

## 1. Protected Names

The following names and designations are protected identifiers:

- Fundacja BBS — Better Balance System
- BBS
- LEO — Logical Epistemic Oversight

Logotypes, graphical symbols, and official visual materials
associated with these names are also protected.

---

## 2. Open Source vs Trademark

The source code in this repository is licensed under
the Apache License 2.0.

However, the Apache License does NOT grant permission
to use the Foundation’s trademarks, brand identity,
or official project names for:

- derivative branding,
- commercial representation,
- public institutional positioning,
- misleading affiliation claims.

---

## 3. Permitted Use

You may:

- Refer to the project descriptively.
- State that your project is based on this repository.
- Use the names in academic or research references.

---

## 4. Restricted Use

You may NOT:

- Claim official affiliation with the Foundation.
- Present derivative versions as official releases.
- Use Foundation branding in commercial marketing.
- Register confusingly similar trademarks.

---

## 5. Official Versions

Only releases explicitly published by
Fundacja BBS — Better Balance System
constitute official versions.

---

## 6. Contact

For trademark usage permissions or clarification,
please contact the Foundation through official channels.
